Setting Fuse Bit Atmega8/16 Dengan Khazama

05.36  Gen Eiryodan  No comments

pertama DOWNLOAD dan instal sofwernya, kemudian buka. 

PERHATIAN : JANGAN SAMPAI SALAH SETTING FUSE. BISA-BISA CHIP TERKUNCI(LOCK) DAN TIDAK BISA DIPROGRAM LAGI. 

>>setting chip untuk atmega8

klik command>>fuses and lock bits 

klik read all 

setting fuse sesuai gambar dibawah ini 
kemudian klik write all 

 >>setting chip untuk atmega16 

klik command>>fuses and lock bits 

klik read all 

setting fuse sesuai gambar dibawah ini
